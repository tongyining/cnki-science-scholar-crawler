**AI-Assisted Literature Collector** 是一个专为研究生和科研人员设计的浏览器插件，用于解决文献综述阶段海量文献检索效率低下的问题。它能根据AI生成的关键词，从 **CNKI (知网)** 和 **Science** 等主流学术数据库中，**自动抓取成百上千篇**相关文献，帮助你将文献调研时间从数小时缩短至十几分钟，是撰写毕业论文和学术论文的强大辅助工具。

![插件界面截图](/images/plugin-screenshot.png)

# 文献智能采集助手

本插件专为论文写作设计，能够快速、批量采集指定领域的学术文献（如知网、Science 等），并通过 AI 辅助分析与整理。

> 传统手动搜索一个领域少则几百篇、多则上千篇文献，效率极低。  
> 使用本插件，**几分钟到十几分钟**即可获取成百上千篇文献，大幅提升文献调研效率。

## 📖 使用教程

### 第一步：生成关键词

让 AI（如 ChatGPT、Claude 等）根据你的论文主题，生成一批**合适的检索关键词**。

> 建议每个检索式为 **2 个关键词组合**：  
> - 1 个 → 搜索结果过多  
> - 3 个 → 搜索结果过少  
> - 2 个 → 最为理想

### 第二步：复制关键词并开始采集

1. 将 AI 生成的关键词（**一行一个**）复制到插件的搜索框中。
2. 点击 **「开始采集」** 按钮。

### 第三步：等待采集完成

- 采集速度约为 **1 分钟处理 10–20 个关键词**（一行一个）。
- 采集完成后，插件会弹出提示。

### 第四步：查看与导出结果

1. 点击 **「查看批次」**。
2. 点击 **「全屏」** 按钮，会在新标签页中打开采集结果。

---

## 📌 导出注意事项

- **自动导出的 CSV 文件** 格式存在特殊处理（为兼容 Excel 计算，避免科学计数法等问题），**不推荐直接使用**。
- **推荐做法**：在全屏查看结果界面，**手动复制**所需内容到本地文件中，格式最稳定。
